0-1 Knapsack Solver

Knapsack solver is a gitHub repository that contains C++ codes about the knapsack 0-1 problem
The generator folder contains two files which create one knapsack 0-1 problem or a pack of problems. We use 320 from these problems and they solvered using six algorithms

Algorithms

  1. 1.Greedy approach
  2. 2.Brute force
  3. 3.Branch and bound
  4. 4.Dynamic programming
  5. 5.Dynamic OR-Tools solver
  6. 6.Integer OR-Tools solver

The knapsack.exe file can create and solve knapsack problems and extract the results to csv files or txt files.In the text files you can see the items which selected from 6 algorithms
